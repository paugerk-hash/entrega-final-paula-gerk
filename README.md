🍰 Pastelería Artesanal
Sitio web responsivo de una pastelería artesanal que ofrece un catálogo de productos con sistema de carrito de compras, recetas, contacto y opciones adaptadas (Sin TACC).

🚀 Características del Sitio
Catálogo de Productos: Galería visual con precios, descripciones y botones de compra.
Carrito de Compras: Enlace integrado a la página de pedidos mediante íconos personalizados.
Diseño Responsivo: Menú adaptado para dispositivos móviles con botón de despliegue interactivo.
Sección de Reseñas: Módulo visual con calificaciones de estrellas y opiniones de clientes reales.
Página de Contacto: Formulario enlazado para la atención al cliente.

🛠️ Tecnologías Utilizadas
Estructura: HTML5 semántico (<header>, <main>, <section>).
Estilos: CSS3 personalizado (styles.css).
Iconografía: Font Awesome Pro (v5.10.0).
Fuentes y Multimedia: Imágenes locales optimizadas para productos y perfiles.

💻 Visualización Local
Para ver y editar este proyecto en tu entorno local:

Clona o descarga este repositorio en tu computadora.
Asegurate de mantener la estructura de carpetas intacta (con las imágenes en img/).
Si usás Visual Studio Code, instalá la extensión Live Preview o Live Server.
Abrí el archivo index.html.
Dale clic derecho y seleccioná "Show Preview" o ejecutá tu servidor local para verlo en: http://127.0.0
